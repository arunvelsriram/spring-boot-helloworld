# helloworld

An example app to capture my learnings on spring boot, Kotlin, Gradle etc.

## Setup

### Running Tests

```
./gradlew testCoverge
```

### Starting Server

```
./gradlew bootRun
```

#### Manual Test
```
http :8080/greeting
http :8080/greeting?name=Arun
```
